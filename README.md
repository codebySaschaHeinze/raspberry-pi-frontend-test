# Fruits Frontend

This repository contains the frontend for the Fruits project.  
It consumes data from the Fruits Backend API and displays fruit information in a simple user interface.

## Tech Stack

- HTML
- CSS
- JavaScript

## Setup

# Clone the repository

git clone <REPO-URL>
cd fruits_frontend

# Open the project

Open index.html in your browser  
(or use a local dev server such as Live Server)

## Backend Connection

The frontend expects the backend API to be running locally.

Default backend URL:
http://127.0.0.1:8000/

Example API endpoint used:

- /api/fruits/

If the backend runs on a different host or port, update the fetch URL accordingly.
